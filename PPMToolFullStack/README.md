
## Description

An API for My Personal Project Management Tool.
 

## Requirements

- spring boot
- database
- marven

 

## Running the api locally

- just fork the Repo and run the main file

 

For connecting the API against the database you can use local database or `docker-compose`
```bash
# database
$ docker-compose up
```
 

## Swagger docs

1. start the api by running the main file
2. visit `http://localhost:9090/actuator/openapi`

Official docs at http://localhost:9090/actuator/swaggerui
